# Приложение для тайминга времени POMODORO 

Приложение для тайминга с авторизацией пользователя и статистикой. Позволяет создать таймер, редактировать, удалять его. Так же собирает статистику пользования таймером.


## Используемые тенологии

**Client:** React, TypeScript, Redux, Redux-Thunk, React-Router

**Server:** Node, Express, PostgreSQL, TypeScript


## Deployment

При запуске приложение автоматически создаёт все необходимые таблицы в БД, если их нет. Подключить базу нужно по образцу в файле .env-sample
